# TODO List App

Aplicación web básica de TODO List desarrollada con Node.js, Express y un frontend en HTML, CSS y JavaScript.

## Funcionalidades
- Crear tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas

## Requisitos
- Node.js v16+
- Git
