##  Instalación

1. Clonar este repositorio:

```
https://github.com/JERM-ED/JERM-ED-To-Do-List.git'

```

# Entrar al proyecto:
```
cd todo-list-node
``` 
 Instalar dependencias:
 ```
 npm install 
 
 ```

Inicia el servidor con:
 ```
node app.js

 ```
 ```
 El servidor correrá en:
http://localhost:3080

```
